# CyberScan

CyberScan is a Python-based security analysis tool focused on:

- Secret detection
- GitHub token exposure
- API key scanning
- Security automation
- DevSecOps analysis

## Features

- Scan local projects
- Detect exposed secrets
- Analyze source code
- Generate security findings

## Technologies

- Python 3
- Regex
- OS module
- Security automation

## Usage

```bash
python3 scanner/secrets_scan.py
```
